# Lifebeat
Приложение помогающее достигать целей и организовывать расписание
# Как запустить
## Linux
Установите зависимости <br><br>
**Debian/Ubuntu**
``` bash
sudo apt update && sudo apt upgrade
sudo apt install sqlite3 libsqlite3-dev
```
Закомпилируйте приложение
```
flutter build linux --release
```
